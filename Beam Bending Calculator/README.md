# Mini Beam Deflection Calculator

This is a simple engineering project that calculates the **deflection of a cantilever beam** under different load values using **Python, NumPy, and Matplotlib**.

It serves as an introductory project to combine programming with engineering fundamentals.

---

## 📌 Beam Deflection Formula

The tip deflection of a cantilever beam loaded with a point force is calculated using:

$$\delta = \frac{F L^3}{3 E I}$$


Where:  
- **F** = Applied force (N)  
- **L** = Beam length (m)  
- **E** = Young’s modulus / Elastic modulus (Pa)  
- **I** = Area moment of inertia (m⁴)  
- **δ** = Tip deflection (m)
